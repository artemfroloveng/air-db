# База данных «Месяц» и «Воздух»

Проект на Python создает SQLite-базу данных для таблиц `months` и `air`, загружает данные о среднемесячных концентрациях бенз(а)пирена в воздухе городов России с сайта НПО «Тайфун» и открывает простой интерфейс для CRUD-операций через Streamlit.

## Что делает проект

- создает базу данных `database.db`;
- создает таблицы `months` и `air`;
- скачивает PDF-справки по воздуху за доступные годы до 2020 года;
- убирает столбцы, где заполнено меньше 60% данных;
- добавляет столбец `date`;
- загружает очищенные данные в SQLite;
- дает интерфейс для добавления, просмотра, изменения и удаления записей.

## Как запустить

### 1. Создать виртуальное окружение

```bash
python -m venv venv
```

### 2. Активировать окружение

Для macOS/Linux:

```bash
source venv/bin/activate
```

Для Windows:

```bash
venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Создать и заполнить базу данных

```bash
python INSTALL_BD.py
```

После этого появится файл:

```txt
database.db
```

### 5. Запустить интерфейс

```bash
streamlit run app.py
```

После запуска Streamlit откроет интерфейс в браузере.

## Таблицы

### `months`

Справочник месяцев.

Поля:

- `id`
- `month_number`
- `month_name`

### `air`

Данные о концентрациях бенз(а)пирена в воздухе.

Поля:

- `id`
- `city`
- `station`
- `year`
- `month_id`
- `date`
- `value`
- `annual_avg`
- `source_url`
- `created_at`
